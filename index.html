One-Time Setup (5 min)

1

Go to console.cloud.google.com → Create a new project

2

Search for Google Sheets API → Enable it

3

Go to APIs & Services → Credentials → Create API Key → Copy it above

4

In your Google Sheet: Share → Anyone with the link → Viewer (for read). For write access, see the OAuth note below.

5

Your Spreadsheet ID is in the URL: docs.google.com/spreadsheets/d/[ID HERE]/edit

⚠

Write access: API keys are read-only. To write back to Google Sheets, you'll need OAuth2. For now, the app uses local storage as primary and exports CSV to paste back into Sheets. A full OAuth version can be built if needed.
